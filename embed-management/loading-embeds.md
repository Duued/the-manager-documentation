# Loading Embeds

To get a list of embeds saved into the bot, type `/embeds list`

<figure><img src="https://i.imgur.com/PLBv83S.png" alt=""><figcaption><p>An example embed</p></figcaption></figure>

To load the embed that's saved to the bot, type `/embed load (number)`

To load from a previously made file, you can upload it by using `/embd file attachment` and drag and drop or use file explorer, or you can leave attachment blank and you will get a prompt to paste into.
